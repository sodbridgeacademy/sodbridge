Course Details

Craft stunning user experiences and bring designs to life with our Frontend Development Bootcamp

Home Courses Course Details

Objectives

Empower students with expertise in building interactive and dynamic web applications using HTML, CSS, JavaScript, and React, focusing on creating intuitive user interfaces and seamless user experiences.

Delve into the world of user-centric design principles, engaging UI/UX practices, and the latest trends in frontend technology, preparing students to create visually stunning and highly interactive web applications. This UI/UX Bootcamp is designed for beginners with a passion for design and technology. To be eligible for this course, candidates should meet the following criteria:

Eligibility

  • Basic Computer Skills: No prior coding experience required, but familiarity with using computers and the internet is recommended.
  • Motivation to Learn: Open to beginners with a strong interest in web development, UI/UX design, and creating visually appealing web interfaces.
  • Interest in Design: Candidates should have a genuine interest in user interface and user experience design, with a willingness to learn and explore creative solutions to design challenges.
  • Access to Required Tools: While not mandatory, having access to a computer with internet connectivity and design software (e.g., Adobe XD, Figma) would be beneficial for practical exercises and assignments.
  • Students and Fresh Graduates: Ideal for students or recent graduates in any discipline seeking to kickstart a career in frontend development or those eager to enhance their tech skills.

Program Outline

Module 1: Introduction to UI/UX

Introduction to UI/UX Design

  • Understanding the basics of User Interface (UI) and User Experience (UX)
  • Differences between UI and UX design
  • Importance of user-centered design
  • Overview of the design process
Module 2: User Research and Personas

User Research and Personas

  • Conducting user research to gather insights
  • Creating user personas based on research
  • Understanding user needs, goals, and behaviors
  • Defining target audience and user scenarios
Module 3: Wireframing and Prototyping

Wireframing and Prototyping

  • Introduction to wireframes: Low-fidelity vs high-fidelity
  • Creating wireframes using Figma
  • Prototyping interactive interfaces
  • Using Figma's prototyping tools to create clickable prototypes
Module 4: Visual Design Principles

Visual Design Principles

  • Typography, color theory, and grid systems
  • Using Figma for layout design
  • Creating consistency in design
  • Designing intuitive and user-friendly interfaces
Module 5: Principles of Interaction Design

Principles of Interaction Design

  • Understanding micro-interactions
  • Using animations and transitions in Figma
  • Designing for user engagement
  • Creating intuitive navigation flows
Module 6: Usability Testing and Iteration

Usability Testing and Iteration

  • Conducting usability testing on prototypes
  • Gathering feedback and analyzing test results
  • Iterating on designs based on feedback
  • Improving usability and accessibility
Module 7: Design Systems and Collaboration

Design Systems and Collaboration

  • Creating and maintaining design systems in Figma
  • Using reusable components for efficiency
  • Collaborating with teams using Figma
  • Version control and managing design updates
Module 8: Collaborative Design Tools and Techniques

Collaborative Design Tools and Techniques

  • Using Figma for real-time collaboration
  • Integrating with other tools (e.g., Slack, Trello)
  • Managing feedback and design reviews
  • Co-designing with stakeholders and developers
Module 9: Developing a Portfolio

Developing a Portfolio

  • Showcasing completed projects in Figma
  • Structuring portfolio for job applications
  • Highlighting design process and decision-making
  • Creating a standout portfolio for UI/UX roles
Module 10: Final Project and Showcase

Final Project and Showcase

  • Capstone project: End-to-end design in Figma
  • Presenting and defending design choices
  • Showcasing the final project to peers and industry professionals
  • Portfolio review and feedback
